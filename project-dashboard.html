Case Study

Hazard Exposure Assessment Dashboard

A personal workflow tool designed to automate hazard exposure assessments for geographically distributed asset portfolios. The project focuses on improving efficiency, consistency, and clarity in spatial risk evaluation.

← Back to Projects Personal project — no proprietary data used

Objective

  • Reduce manual steps when assessing large portfolios of assets.
  • Produce consistent exposure metrics and spatial summaries.
  • Create a repeatable workflow for hazard exposure analysis.

Approach

  • Import asset coordinates and hazard layers.
  • Automate spatial overlay analysis.
  • Generate charts and maps summarizing exposure.

Tools

  • Streamlit workflow interface
  • GIS spatial overlays
  • Python preprocessing scripts

Role

  • Designed the entire workflow architecture.
  • Defined exposure indicators and outputs.
  • Developed the tool as a personal workflow optimization.
