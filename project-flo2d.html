2D Flood Modeling Using FLO-2D

Built two-dimensional hydrodynamic flood simulations to model river flooding using FLO-2D within a GIS workflow. The model produced flood depth and extent outputs used to generate hazard maps for interpretation and planning.

Objective

  • Simulate river flooding behavior under specified hydrologic conditions.
  • Generate flood depth/extent maps suitable for hazard interpretation.
  • Convert model outputs into GIS layers that can support exposure analysis.

Approach

  • Prepare terrain and model grid inputs (DEM processing and conditioning).
  • Define hydrologic and boundary conditions required for simulation.
  • Run FLO-2D and post-process results into map-ready datasets.

Tools Used

  • FLO-2D (2D hydrodynamic modeling)
  • GIS platform for pre/post-processing (QGIS/ArcGIS)
  • DEM + supporting spatial layers

What I Owned (Leadership)

  • Built the model workflow and ensured GIS-ready outputs.
  • Standardized post-processing steps for consistent mapping.
  • Translated results into clear hazard visuals for stakeholders.

  • GIS-integrated workflows make it easier to convert results into exposure-ready layers.
