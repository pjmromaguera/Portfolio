River Turbidity Assessment Using Remote Sensing

Used Google Earth Engine (GEE) and satellite remote sensing to assess turbidity patterns in rivers around Metro Manila. Outputs supported identification of potential siltation zones for planning and prioritization considerations.

Objective

  • Map turbidity indicators and sediment concentration patterns.
  • Identify potential hotspots consistent with siltation zones.
  • Summarize results in visuals suitable for planning discussions.

Approach

  • Process satellite imagery in GEE (cloud masking and compositing).
  • Derive turbidity indicators using spectral indices.
  • Analyze spatial patterns and time-series changes to flag hotspots.

Tools Used

  • Google Earth Engine
  • Satellite imagery (e.g., Sentinel-2 / Landsat)
  • Remote sensing indices + time-series summaries

What I Owned (Leadership)

  • Designed the analysis workflow and index selection for turbidity interpretation.
  • Built the processing pipeline in GEE to make results repeatable.
  • Translated outputs into maps/graphics suitable for non-technical audiences.
