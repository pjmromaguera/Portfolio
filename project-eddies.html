Ocean Eddy Detection and Lifecycle Tracking (Freelance)

Processed multi-year oceanographic NetCDF datasets to detect mesoscale eddies and analyze sea surface anomalies. Extracted eddy-relevant parameters (e.g., temperature structure and flux-related indicators) and tracked eddy lifecycles over time.

Objective

  • Identify eddies and characterize associated ocean circulation anomalies.
  • Extract relevant physical parameters from gridded NetCDF datasets.
  • Track eddy lifecycle (formation, movement, dissipation) to support interpretation.

Approach

  • Process and subset multi-year NetCDF data for the area/time of interest.
  • Apply a Python-based eddy detection library/workflow.
  • Track eddy features through time and summarize trajectories and persistence.

Tools Used

  • Python scientific stack (xarray, NumPy, etc.)
  • NetCDF gridded datasets
  • Specialized library for eddy detection / tracking

Responsibilities

  • End-to-end execution: data processing, feature detection, tracking, and summarization.
  • Defined parameters/outputs needed for eddy characterization.
  • Delivered structured outputs suitable for further interpretation by the client.

Key Takeaways

  • Feature tracking enables analysis of persistence, movement paths, and evolution of anomalies.
  • NetCDF-based workflows scale well for multi-year gridded datasets.
  • Parameter extraction supports richer interpretation beyond “eddy/no eddy”.
