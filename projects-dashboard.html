Hazard Exposure Assessment Dashboard (Personal Project)

A personal workflow tool built to speed up and standardize hazard exposure assessments for geographically distributed asset portfolios. This project is independent and does not include proprietary client systems or confidential data.

Objective

  • Reduce manual steps in exposure assessment when asset counts range from tens to hundreds of sites.
  • Produce decision-ready outputs: quantitative exposure metrics + clear visual summaries.
  • Improve consistency across assessments by using a repeatable workflow.

Approach

  • Ingest asset locations and selected hazard layers.
  • Automate spatial overlays and summarization into exposure indicators.
  • Generate maps/charts to support interpretation of spatial risk patterns.

Tools Used

  • GIS + spatial analytics workflow
  • Python (optional for preprocessing/automation)
  • Map and chart outputs for reporting

What I Owned (Leadership)

  • Led workflow design and implemented a standardized assessment process.
  • Defined exposure metrics and output formats to reduce rework.
  • Built the project as a personal optimization initiative (no client delivery).
